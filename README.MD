## Project 설명
[네이버 Maps Open API](https://guide.ncloud-docs.com/docs/naveropenapiv3-maps-overview)를 사용한 프로젝트입니다.  

### 기능 설명
- 학교 주변의 가게들을 지도를 통해 조회할 수 있다.
- 누락된 가게가 있다면 추가할 수 있다.
- 가게의 상세 정보를 확인할 수 있다.
- 가게의 별점을 조회하고, 등록할 수 있다.
- 키워드로 가게를 검색할 수 있다.
